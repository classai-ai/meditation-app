# 마음챙김 명상 웹앱

기획안과 소스 문서를 기반으로 만든 Next.js 명상 서비스 MVP입니다.

## 실행 방법

```bash
cd meditation-app
npm install
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 을 엽니다.

## 주요 기능

- 시간대별 배경과 추천 명상
- 3/5/10/15분 빠른 명상 시작
- 유튜브 임베드 명상 재생
- localStorage 기록 저장
- 월간 달력과 통계
- 7일 마음 챙김 챌린지

## 배포

Vercel에 `meditation-app` 디렉터리를 연결해 배포할 수 있습니다.
